import { ShowcaseCTA } from 'starlight-showcases'

## Call to action (CTA)

To encourage users to contribute to your showcase page, use the [`<ShowcaseCTA>`](/components/call-to-action/) component.

<ShowcaseCTA>
  Have you built a showcase page? [Open a PR](#call-to-action-cta) to add it to
  the list!
</ShowcaseCTA>
